Calculadora Juros Compostos

Prática de Lógica de Programação

Softwares Usados

Calculadora financeira desenvolvida para análise de contratos bancários.

O foco técnico deste projeto foi a implementação de fórmulas matemáticas em JavaScript para garantir a precisão da capitalização ao longo do tempo.

O projeto utiliza manipulação de DOM para entrega de resultados em tempo real, facilitando a simulação de diferentes cenários financeiros.

Calculadora Comprometimento de Renda

Prática de Lógica de Programação

Softwares Usados

Sistema de cálculo baseado na Lei do Superendividamento.

A lógica foi estruturada para analisar a margem de subsistência do usuário, calculando automaticamente a porcentagem da renda comprometida com dívidas bancárias.

Tecnicamente, o desafio foi criar um formulário dinâmico que valida diferentes tipos de despesas, garantindo que o resultado final esteja em total conformidade com os parâmetros legais vigentes.

Calculadora de Juros Abusivos

Prática de Lógica de Programação

Softwares Usados

Ferramenta técnica para identificação de irregularidades em contratos de financiamento.

Desenvolvido com HTML5, CSS3 e JS puro, o sistema processa os dados inseridos e gera um feedback visual imediato sobre a viabilidade de uma ação revisional.
